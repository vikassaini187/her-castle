This plugin will show the popup when Contact Form 7 has been submitted. You can customize popup title, message and colors from backend.
== Description ==

Popup for Contact Form 7 plugin will display the fully customizable and beautiful popup when contact form 7 is submitted. you can customize text, font-color, popup size etc.

Contact Form 7 Popup features: 

* Shows popup when form submitted successfully.
* You can customize popup content, background and font color.
* Popup is fully responsive and compatible in all major browser.

== Installation ==

1. Install and Activate the plugin through the 'Plugins' menu in WordPress
2. You can change settings inside "Contact >> Popup" in wordpress admin.

== Frequently Asked Questions ==

= Can I set different popup for individual contact form? =
No, Currently this plugin supports same popup template for all forms, however you can enable/disable popup on specific form.

= Can I customize Popup design? =

Yes, You can customize popup text, design under Contact >> Popup settings menu.
